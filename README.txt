Бот пересылает сообщения пользователя администраторам. 
Администратор может ответить на сообщение у себя, 
в таком случае пользователь получит от администратора ответ.

В ответ на команду /start пользователь получает иерархическое 
меню, описанное в БД в контрольной панели. Результат ответов 
также отправляется администраторам


Настроено на использование sqlite. 
Для запуска нужно поставить переменную окружения c абсолютным путем к файлу базы данных.
Общий файл для двух проектов

    export DJANGO_DATABASE_NAME=/path/to/your/sqlite/database/file/db.sqlite3
